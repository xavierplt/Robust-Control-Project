### Prerequisites
* **MATLAB** (Required for the code base)
* **MATLAB Control System Toolbox** * **MATLAB Robust Control Toolbox** (Crucial for the H-infinity and mu-Synthesis functions: `hinfsyn` and `musyn`).

### Files
**`Code_toolbox.m`:** The main script containing the system definitions, weighting function setup, $H_{\infty}$ synthesis, $\mu$-Synthesis, and time-domain simulations.


### Execution
1.  Open **MATLAB**.
2.  Ensure the **Control System** and **Robust Control** Toolboxes are installed.
3.  Run the **`Code_toolbox.m`** script. The script automatically executes the entire synthesis and generates the Bode and time-domain plots.
